# 6. Сборка и запуск

Этот документ описывает, как запустить фронтенд-приложение для локальной разработки и как собрать его для развертывания.

## 1. Среда разработки (Vite)

Проект использует **Vite** в качестве инструмента для сборки и сервера для разработки. Vite обеспечивает чрезвычайно быструю "горячую" перезагрузку модулей (HMR) и оптимизированный процесс сборки.

-   **Сервер для разработки**: `vite` запускает локальный сервер, который компилирует TypeScript/TSX "на лету" и обслуживает ваше приложение.
-   **Сборка для продакшена**: `vite build` компилирует, транспилирует и минифицирует все ваши файлы в оптимизированный набор статических ассетов (HTML, JS, CSS) в папке `dist`.

## 2. Требования

1.  **Запущенный бэкенд**: Убедитесь, что бэкенд на Python запущен и доступен по адресу `http://127.0.0.1:8000`.
2.  **Node.js и npm**: Необходимы для установки зависимостей и запуска скриптов.

## 3. Процесс запуска для локальной разработки

1.  **Откройте терминал** в **корневой папке проекта** (там, где находится файл `package.json`).

2.  **Установите зависимости** (если вы делаете это в первый раз):
    ```bash
    npm install
    ```

3.  **Запустите сервер для разработки**:
    ```bash
    npm run dev
    ```

4.  **Откройте браузер**:
    -   После запуска Vite выведет в консоль локальный URL вашего приложения (обычно `http://localhost:5173`).
    -   Откройте этот адрес в вашем браузере.

5.  **Настройте окружение API**:
    -   При первом запуске откроется страница входа.
    -   Найдите переключатель "Окружение API" и выберите **"Локальный"**.
    -   Страница перезагрузится, и приложение будет готово к работе с вашим локальным бэкендом.

## 4. Процесс сборки для развертывания (Production)

1.  **Убедитесь, что все зависимости установлены**: `npm install`.
2.  **Запустите команду сборки**:
    ```bash
    npm run build
    ```
3.  **Результат**:
    -   Vite создаст папку `dist` в корне проекта.
    -   Эта папка содержит все необходимые статические файлы для развертывания на любом хостинге (например, Yandex.Cloud S3).
    -   Подробная инструкция по развертыванию находится в файле `DEPLOYMENT_FRONTEND.md`.
