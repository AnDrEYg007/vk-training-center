
# Статистика и Графики (System Lists)

**Компонент:** `features/lists/components/ListStatisticsPanel.tsx`
**API:** `services/api/lists.api.ts`

## 1. Общая концепция

Панель статистики предоставляет визуальный анализ данных. Для разных типов списков отображаются разные наборы виджетов.

## 2. Виджеты "Пользователи"

Для списков людей (Подписчики, История, Активности) доступны следующие метрики:

### Демография
*   **Возраст (Столбчатая диаграмма):** Распределение пользователей по возрастным группам (`<16`, `16-20` ... `45+`). Рассчитывается на бэкенде на основе поля `bdate`.
*   **Дни рождения:** График именинников по месяцам. Полезно для планирования поздравлений.

### Технические метрики
*   **Платформы (Прогресс-бары):** Показывает, с каких устройств пользователи сидят в VK (Android, iPhone, Web, m.vk). Данные берутся из поля `last_seen.platform`.
*   **География (Pie Chart):** Топ городов.

## 3. Аналитика Рассылки (Mailing)

Для списка "В рассылке" доступны уникальные метрики, основанные на анализе диалогов.

### График: Динамика новых диалогов (First Contact)
*   **Ось X:** Время (День/Месяц).
*   **Ось Y:** Количество **новых** диалогов, начатых в этот период.
*   **Назначение:** Позволяет оценить эффективность рекламных кампаний или приток лидов в ЛС. Строится на основе поля `first_message_date`.

### Виджет: Последний контакт (Last Contact)
Классифицирует базу по "теплоте" (времени с момента последнего сообщения):
*   `Сегодня`, `3 дня`, `Неделя` — Горячая база.
*   `> 1 месяца`, `> 3 месяцев` — Остывающая база.
*   `> Года` — Мертвые души.

### Виджет: Доступность ЛС
*   **Целевая аудитория:** "Активные" пользователи (не забаненные), которым "Можно писать" (`can_access_closed = true`).

## 4. Статистика Постов

*   **Динамика:** График публикаций и реакций (лайки, просмотры) во времени.
*   **Топы:** Ссылки на посты с максимальным количеством лайков, репостов, комментариев и просмотров.
