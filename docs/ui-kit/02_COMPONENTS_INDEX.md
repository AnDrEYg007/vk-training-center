# 2. Индекс компонентов

Этот файл служит оглавлением для всех переиспользуемых UI-компонентов в приложении. Каждый компонент имеет отдельную страницу с подробным описанием его структуры, состояний и правил использования.

## Принципы и Гайдлайны

-   **[Принципы UI/UX и поведения](./04_UI_UX_PRINCIPLES.md)**
    -   Описание: Фундаментальные правила защиты данных, валидации, работы с модальными окнами и экономии пространства.

## Основные элементы

-   [Кнопки](./components/BUTTONS.md)
    -   Описание: Основной элемент для выполнения действий. Включает несколько вариантов (основные, вторичные, деструктивные) и состояний.

-   [Карточки](./components/CARDS.md)
    -   Описание: Контейнеры для отображения контентных единиц, таких как посты и заметки.

-   [Модальные окна](./components/MODALS.md)
    -   Описание: Всплывающие диалоговые окна для редактирования, подтверждения действий или просмотра информации.

-   [Поля ввода и контролы](./components/INPUTS_AND_CONTROLS.md)
    -   Описание: Элементы форм, такие как текстовые поля, переключатели и фильтры.

-   [Таблицы](./components/TABLES.md)
    -   Описание: Отображение и редактирование наборов структурированных данных.

## Визуализация данных

-   **[DiffViewer (Сравнение текста)](./components/VISUALIZATION.md)**
    -   Описание: Компонент для визуального отображения разницы между двумя версиями текста (старый vs новый) с цветовой подсветкой добавлений и удалений.

## Навигация и структура

    -   Описание: Ключевые компоненты, формирующие основной макет приложения и обеспечивающие навигацию.

-   **[Страница обучения](./TRAINING_PAGE.md)**
    -   Описание: Структура и логика работы раздела "Центр обучения".

## Индикаторы состояний

-   [Скелетоны и Лоадеры](./components/SKELETONS_AND_LOADERS.md)
    -   Описание: Элементы, информирующие пользователя о процессах загрузки данных или выполнения фоновых задач.

## Графика
