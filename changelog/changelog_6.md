# Журнал изменений (Changelog) - Часть 6

Этот файл содержит историю версий и описание основных изменений в приложении "VK Content Helper", начиная с версии 1.67.0.

## Версия 1.67.0 - 2024-08-06

-   **Улучшение надежности: "Бронебойная" система работы с токенами**
    -   **Что внедрено:** Реализованы продвинутые стратегии для обеспечения бесперебойной работы приложения даже при частичных проблемах с правами доступа или блокировками аккаунтов.
    -   **Стратегия "Broadcast" (для чтения):** При обновлении "закрытых" разделов (Отложенные и Предложенные посты), система теперь не полагается на один токен. Она одновременно опрашивает **ВСЕ** доступные токены (системные аккаунты + токен из .env) и выбирает тот ответ, в котором больше всего данных.
        -   *Что дает:* Если основной сервисный аккаунт потерял права админа, но в системе есть другой аккаунт с правами, данные все равно загрузятся.
    -   **Стратегия "Fallback" (для записи):**
        -   **Публикация:** При создании нового поста приоритет отдается токену сообщества (`communityToken`). Если он не работает, система автоматически пробует токены пользователей.
        -   **Редактирование:** При изменении поста система перебирает все токены, пока не найдет тот, у которого есть права на редактирование этого конкретного поста (авторство или админские права).
    -   **Что дает пользователям:** Максимальная отказоустойчивость. Ошибки "Access denied" теперь будут возникать только в самом крайнем случае, когда ни один из подключенных аккаунтов не имеет доступа к группе.
